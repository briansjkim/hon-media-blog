## Installation

#### Clone Project

```shell
git clone https://github.com/briansjkim/hon-media-blog
```

#### Install dependencies

```shell
npm install
```

#### Run Gatsby

```shell
npm run develop
```

Navigate to `http://localhost:8000/`. The app will automatically reload if you change any of the source files.
